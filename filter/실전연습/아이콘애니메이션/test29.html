폰트어썸(FontAwesome) 사용법

        연결법
        

            <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesme/5.11.2/css/all.css">
        

        해당 아이콘 연결
        

            <i class="fa fa-camera-retro">
        

            치트시트 : https://fontawesome.com/cheatsheet?from=io


글리프 아이콘(Glyphicons) 사용법

        연결법
        

            <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap.min.css">
        

        해당 아이콘 연결
        

            <span class="glyphicon glyphicon-asterisk">
        

            치트시트 : https://glyphicons.bootstrapcheatsheets.com/

        연결법
        

            <script src="https://unpkg.com/ionicons@4.5.10-0/dist/ionicons.js"> </script>  

            <https://unpkg.com/ionicons@4.5.10-0/dist/css/ionicons.min.css>
        

        해당 아이콘 연결
        

            <ion-icon name="heart">
            <i class="ion ion-md-beer">
